GitCMS

Kelola konten langsung dari repository GitHub Anda. Tanpa database, tanpa server.

Token disimpan lokal di browser Anda, tidak pernah dikirim ke pihak ketiga.
Cara membuat token →
  1. Buka GitHub → Settings → Developer settings
  2. Pilih Personal access tokens → Fine-grained tokens
  3. Klik Generate new token
  4. Beri akses ke repository CMS Anda
  5. Pada Repository permissions, set Contents ke Read and write
  6. Salin token, lalu tempel di atas
Data Anda tetap menjadi milik Anda — semua tersimpan di repo GitHub.

Tema

Pilih tampilan situs. Setiap tema mengatur tampilan publik sepenuhnya — atur isinya lewat menu Sesuaikan.

Menambah tema baru: cukup unggah folder ke themes/<nama>/ berisi theme.json (lengkap dengan blok customize) dan templatnya. Tema otomatis muncul di galeri ini — tanpa mengubah kode inti.

Tema aktif tidak mendeklarasikan blok customize di theme.json, sehingga tampilannya sepenuhnya statis. Aktifkan tema lain dari menu Tema, atau tambahkan skema customize pada theme.json tema ini.
